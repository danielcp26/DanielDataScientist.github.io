Nashville Housing Data Cleaning with PostgreSQL

This project focuses on cleaning and transforming real estate transaction data using PostgreSQL, ensuring data accuracy and consistency. Key tasks include handling missing values with COALESCE(), splitting addresses into street, city, and state using SPLIT_PART(), standardizing categorical values with CASE WHEN, removing duplicates using ROW_NUMBER(), and dropping unnecessary columns to streamline the dataset. Utilizing CTEs, string functions, and aggregate queries, this project prepares the data for further analysis and visualization.
